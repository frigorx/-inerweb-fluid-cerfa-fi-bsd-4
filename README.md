# inerWeb Fluide

[![GitHub Pages](https://img.shields.io/badge/demo-GitHub%20Pages-blue)](https://inerweb.github.io/fluide/)

Application de traçabilité des fluides frigorigènes et génération CERFA 15497*04.

## 🚀 Déploiement

### Frontend (cette page)
Hébergé sur GitHub Pages.

### Backend
Google Apps Script - Voir documentation séparée.

## 📱 Fonctionnalités

- ✅ Traçabilité fluides frigorigènes F-Gas
- ✅ Génération CERFA 15497*04
- ✅ Mode Formation / Officiel
- ✅ Multi-site / Multi-atelier
- ✅ Responsive (mobile, tablette, PC)
